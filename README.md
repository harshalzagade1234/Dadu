# अहमदनगर जिल्हा माध्यमिक शिक्षक संघ

अहमदनगर जिल्हा माध्यमिक शिक्षक संघ (Ahmednagar Zilla Madhyamik Shikshak Sangh) यांची माहितीपत्रक वेबसाईट.

## रचना

- `index.html` — संपूर्ण वेबसाईट (एकाच फाईलमध्ये HTML + CSS + JS)
- `images/` — अध्यक्ष, पदाधिकारी व बातम्यांचे फोटो

## Local मध्ये पाहण्यासाठी

फक्त `index.html` ब्राउझरमध्ये उघडा, किंवा:

```bash
npx serve .
```

## Deploy (Vercel)

हे रिपॉझिटरी Vercel ला जोडलेले आहे — `main` branch वर push केल्यावर आपोआप deploy होईल.
